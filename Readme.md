  _beta_.

## Installation

    $ component install yields/cycle

## API

#### Cycle(el)

Initialize `Cycle` with `el`.

#### bind()

Bind internal events

#### unbind()

Unbind internal events

#### select(el)

Select specific option with `el` or option `index`.

#### up()

Cycle upwards.

#### down()

Cycle downwards.
